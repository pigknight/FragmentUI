# FragmentUI
一个基于Fragment的动态构建的快速UI流开发类,方便快速构建App的UI Flow.

一个典型的音乐Demo的UI架构如下:

Layer-1: --Root

Layer-2: --|--Content

Layer-3:------|--Category

Layer-3:------|--AlbumList

Layer-3:------|--MusicList

Layer-2: --|--Playing
